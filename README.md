# Weather Data ETL Pipeline

## Project Overview

This project demonstrates an end-to-end ETL pipeline using Python, PostgreSQL, SQL, and Power BI.

## Architecture

Weather API
↓
Python Extraction
↓
Data Transformation (Pandas)
↓
PostgreSQL Database
↓
Power BI Dashboard

## Tech Stack

- Python
- Pandas
- PostgreSQL
- SQL
- Power BI

## Features

- Extract weather data from API
- Clean and transform data
- Load data into PostgreSQL
- Perform SQL analysis
- Build interactive Power BI dashboard
